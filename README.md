# Enterprise Agentic RAG Assistant

A production-style Retrieval-Augmented Generation (RAG) assistant built from scratch while learning every concept step by step.

## Goals

- Learn RAG from fundamentals
- Build a production-ready pipeline
- Understand every component instead of copying code
- Implement hybrid search, reranking, LangGraph, and FastAPI

## Tech Stack

- Python
- LangChain
- LangGraph
- ChromaDB
- OpenAI / Gemini
- BGE Embeddings
- Cross Encoder Re-ranking
- FastAPI
